무지예뽀 쇼핑몰은 한국형 쇼핑몰 운영에 최적화된 워드프레스 플러그인입니다.

== Description ==

무지예뽀 쇼핑몰 플러그인은 간단하고 직관적인 한국형 쇼핑몰 솔루션입니다.

= 주요 기능 =

* 상품 관리 시스템
* 장바구니 기능
* 찜하기 기능
* 상품 카테고리 관리
* CSV 대량 상품 등록
* 반응형 디자인
* AJAX 기반 빠른 응답
* 포인트 시스템
* 쿠폰 시스템

= 특징 =

* 가볍고 빠른 성능
* 한국형 쇼핑몰에 최적화
* 모바일 친화적
* SEO 최적화
* 보안 강화

== Installation ==

1. 플러그인 파일을 `/wp-content/plugins/muziyeppo-shop` 디렉토리에 업로드합니다.
2. 워드프레스 관리자 페이지의 '플러그인' 메뉴에서 플러그인을 활성화합니다.
3. '무지예뽀 상품' 메뉴가 생성됩니다.
4. 설정 페이지에서 기본 설정을 구성합니다.
